I'll write a simple login form code in HTML, CSS, and JavaScript.

**Login Form HTML**

```html
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Login Form</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h2>Login Form</h2>
        <form id="login-form">
            <label for="username">Username:</label>
            <input type="text" id="username" name="username"><br><br>
            <label for="password">Password:</label>
            <input type="password" id="password" name="password"><br><br>
            <button id="submit-btn">Login</button>
        </form>
        <p id="error-message"></p>
    </div>

    <script src="script.js"></script>
</body>
</html>
```

**Login Form JavaScript**

```javascript
// script.js
document.getElementById("submit-btn").addEventListener("click", (e) => {
    e.preventDefault();
    const username = document.getElementById("username").value;
    const password = document.getElementById("password").value;
    const errorMessage = document.getElementById("error-message");

    if (username === "" || password === "") {
        errorMessage.innerText = "Please enter both username and password.";
    } else {
        // Send login request to server or API
        // For demonstration purposes, we'll just log the input values to the console
        console.log(`Username: ${username}, Password: ${password}`);
        errorMessage.innerText = "";
    }
});
```

**Explanation**

1. **HTML**: The HTML code creates a simple login form with two input fields (username and password) and a submit button. The form is wrapped in a container div with a CSS class `container`.
2. **CSS**: The CSS code styles the login form using various properties like `font-family`, `background-color`, `border`, `border-radius`, and `box-shadow`. It also defines styles for the label, input fields, and submit button.
3. **JavaScript**: The JavaScript code adds an event listener to the submit button. When the button is clicked, it prevents the default form submission behavior and checks if both username and password fields are filled. If not, it displays an error message below the form. If both fields are filled, it logs the input values to the console for demonstration purposes.

Note: In a real-world application, you would replace the console.log statement with a server-side request to authenticate the user.
